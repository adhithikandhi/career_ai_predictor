Smart Career AI Predictor
Overview

Smart Career AI Predictor is an AI-powered career guidance web application built using Python and Streamlit. The application analyzes uploaded resumes in PDF format and provides intelligent insights including job role recommendations, salary estimation, skill gap analysis, and location-based job suggestions.

The system evaluates user skills against industry requirements and generates personalized career recommendations.

Key Features
Resume Analysis
Extracts and processes text from uploaded PDF resumes
Identifies technical and professional skills automatically
Matches skills with selected career sector
Career Dashboard
Displays overall skill match percentage
Shows matched skills count
Shows missing skills count
Job Recommendation System
Supports multiple domains including IT, Data Science, Finance, Marketing, HR, and Design
Provides Remote, Hybrid, and Office job categories
Groups job opportunities based on cities
Salary Prediction
Predicts annual salary based on skill match score
Provides estimated monthly salary
Displays salary range per industry sector
Location-Based Job Mapping
Suggests job opportunities based on selected cities
Covers major job markets such as Bangalore, Hyderabad, Chennai, Mumbai, Pune, and Delhi
Skill Gap Analysis
Identifies missing skills from resume
Provides structured improvement recommendations

Installation and Setup
1. Clone the Repository
git clone https://github.com/adhithikandhi/career_ai_predictor.git
cd career_ai_predictor
2. Install Dependencies
pip install -r requirements.txt
3. Run Application
streamlit run app.py
